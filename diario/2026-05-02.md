### Lições aprendidas

- **YOUTUBE_UPLOAD_VIDEO exige videoFilePath LOCAL** — URLs S3 do Gemini são para download, não para upload direto
- **Composio CLI login**: se o flow OAuth expirar, copiar user_data.json de outro agente com a mesma API key resolve
- **YAML indentação Composio**: x-consumer-api-key DEVE estar dentro de headers (6 espaços), não como sibling (4 espaços)
- **Telegram caption**: limite de 1024 chars para sendPhoto. Se o texto for longo, usar caption curto + mensagem separada
